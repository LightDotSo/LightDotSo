## Show

1. **Cross-chain Swaps** - The user puts swapping assets spanning multiple chains into one cart.
2. **Cart Checkout** - The user checks out a cart of items using the Light Protocol.
3. **Execution of Transactions** - The user executes transactions on the Light Protocol, paying tokens on one chain.
4. **Receipt** - The user receives a receipt of the transaction(s), directing to the Light Explorer for more details.
5. **Refund** - The user can refund the surplus escrowed tokens back from the billing settings (or Light Protocol Escrow).
