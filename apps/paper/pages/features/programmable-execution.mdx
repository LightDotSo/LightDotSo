# Programmable Execution in Light

![programmable-execution](/programmable-execution.png)

:::tip[Programmable Execution]
Programmable Execution empowers users to set intents that orchestrate a suite of transactions within the cross-chain environment.
Acknowledging the isolated nature of individual blockchain states, Lightintroduces a programmable execution layer that facilitates the asynchronous execution of transactions, seamlessly bridging these disparate chains.
:::
