# Programmable Execution in Light

![programmable-execution](https://assets.light.so/paper/programmable-execution.png)

:::tip[Programmable Execution]
Programmable Execution empowers users to set intents that orchestrate a suite of transactions within the cross-chain environment.
Acknowledging the isolated nature of individual blockchain states, Light introduces a programmable execution layer that facilitates the asynchronous execution of transactions, seamlessly bridging these disparate chains.
:::
