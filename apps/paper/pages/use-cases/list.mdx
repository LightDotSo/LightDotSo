# List of Use Cases only possible with Light

A list of things that Light enables that is not possible with other bridges/wallets:

### [Batch Send ETH](/use-cases/batch-send)

Batch send ETH with a single signature.

### [USDC Selection](/use-cases/usdc-selection)

Automatically route the underlying USDC to the corresponding chain with the lowest fees.

### [Transaction Graph](/use-cases/transaction-graph)

Execute complex workflows across multiple chains in a single transaction.
