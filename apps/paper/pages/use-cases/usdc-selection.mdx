# USDC Selection

![usdc-selection](https://assets.light.so/paper/usdc-selection.png)

Right now, there are many types of USDC across many different bridges and chains.
Through [Light Programmable Intents](https://assets.light.so/paper/light-programmable-intents), we can create a single interface to select the USDC that automatically routes for the user's needs.
